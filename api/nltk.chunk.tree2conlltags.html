nltk.chunk.tree2conlltags¶

nltk.chunk.tree2conlltags(t)[source]¶

Return a list of 3-tuples containing (word, tag, IOB-tag). Convert a tree to the CoNLL IOB tag format.

Parameters

t (Tree) – The tree to be converted.

Return type

list(tuple)
